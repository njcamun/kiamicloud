# KiamiCloud — apps Cloud (Cloudflare)

Cliente Flutter para a API **Cloudflare Workers** (beta por defeito).

| Plataforma | Pasta | Pacote |
|------------|-------|--------|
| Android | `mobile/` | `kiamicloud_mobile` |
| Web | `web/` | `kiamicloud_web` |
| Windows | `desktop/` | `kiamicloud_desktop` |

```powershell
cd apps/cloud/mobile
flutter pub get
flutter run
```

Variante **ZimaBlade (LAN)**: Definições → Mudar de servidor (admin deve activar a permissão).
